摩尔投票法

求众数

题目描述

给定一个大小为 n 的数组，找到其中的众数。众数是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在众数。

解题思路

一个数字$a$ 如果在整个数组中出现的次数大于$\frac{n}{2}$ 次(注意这里是大于)，那么假如把数组分成很多的小块$b_i, i =1, 2, 3,…,d$，每一块中数字$a$ 都占据了块长的一半，那么最后的那个块$b_d$里面所含有的一定只有$a$。而假如在前面的$b_i, i=1,2,3,..,d-1$块中，数字$a$没有占到块长的一半，那么最后的那个块$b_d$则也一定只有$a$

题目

Given an integer array of size n, find all elements that appear more than ⌊ n/3 ⌋ times. The algorithm should run in linear time and in O(1) space.

解题思路

思想和前一题类似。

可能性1

假如符合题目条件的数有2个，分别为$a$ 和$b$, 那么把数组分成很多的小块$b_i, i =1, 2, 3,…,d$，每一块中数字$a$ 和$b$都占据了$\frac {1}{3}$。 最后的那一块$b_d$ 一定只包含了$a$ 和 $b$ 。那么假如前面的块$b_i, i =1,2,3,…,d-1$ 中$a$ 和$b$ 都没有占到$\frac{1}{3}$, 那么最后的块$b_d$一定只包含了$a$ 和$b$ 。

可能性2

符合题目条件的数只有1个，为a。同样把数组分成很多的小块$b_i, i =1, 2, 3,…,d$，每一块中数字$a$ 都占据了$\frac {1}{3}$, 那么最后的那一块$b_d$ 一定只包含了$a$ 。假如前面的块$b_i, i =1,2,3,…,d-1$ 中$a$ 都没有占到$\frac{1}{3}$, 那么最后的块$b_d$一定只包含了$a$ 。

最后判定

可以发现，上述中$a$是一定为解的，而$b$不一定是解，除非最后只得到一个数字，其他情况下我们无法确定最后得到的两个数字哪个是$a$, 哪个是$b$, 因此需要重新遍历一遍数组，判断最后剩余的两个数字是否符合题述要求

程序评价

程序实现的巧妙之处在于，对$res1$和$res2$用if else if 语句进行了优先级的划分。若没有$res1$, 则添加$res1$, 其次才是添加$res2$, 在两个数都添加完之后，才开始添加”其他数”（那些零散的，总数目占一个$b_i$ 长度$\frac{1}{3}$）的数。
